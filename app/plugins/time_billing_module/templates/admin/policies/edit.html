Policies adjust wage rate / pay for timesheet shifts (weekends, bank holidays, night window, optional clock-split night, JSON time bands, overtime tiers). Overtime (shift) uses the hours on each line: hours up to the first threshold are paid at the effective rate; beyond that, the tier multiplier applies.

How this maps to a timesheet line
For Overtime (per shift), the system uses actual start / end minus break as shift hours. Hours up to First threshold (hours) use the normal chosen rate (after weekend/BH/night). Hours beyond that use Tier 1 multiplier on the rate. If you set a second threshold and tier-2 multiplier, hours beyond the second threshold use tier 2.
