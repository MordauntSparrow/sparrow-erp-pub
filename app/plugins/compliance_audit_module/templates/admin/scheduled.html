Scheduled assurance exports

Each schedule runs once per hour when the current UTC hour matches the time you set (useful for a fixed “overnight” slot). Every run builds a ZIP evidence pack (PDF summary, spreadsheet, and machine-readable copy) and stores it in your site’s uploads folder. Packs older than about three days are removed automatically.

Automatic runs are listed in the export log without a personal PIN step-up, because no individual is clicking export at that moment—this is expected for scheduled jobs. Your organisation can stop the background scheduler via server configuration if required.
