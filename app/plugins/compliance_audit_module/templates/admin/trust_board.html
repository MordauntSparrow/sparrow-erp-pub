Operational pointers for board packs. Not legal advice. Confirm with your governance lead and counsel.

How this module supports accountability
  • Unified timeline — CAD, comms, triage, ePCR case rows, JSON signals, MI, training audit, identity log, admin audit, prior exports.
  • Exports — CSV / JSON / PDF / ZIP evidence pack; manifest + SHA-256; PIN step-up for interactive downloads.
  • Scheduled ZIPs — file-based retention with automatic purge (default ~72h on disk).
  • Gap report — structural checks on cases.data (not clinical adequacy).
