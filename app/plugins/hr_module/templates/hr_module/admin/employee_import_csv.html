Import employees from CSV

Upload a CSV file, then map each column to the right employee field. One import can create new people and fill HR profile details, or update existing records when you enable that option below. Email must be mapped — it is how we match and de-duplicate people. For new rows, name is required. Time Billing role is optional; if you leave it blank, a sensible default is applied and you can adjust it later. New employees receive a welcome email with a temporary password and a link to the employee portal (your organisation must have email delivery set up on the server). Back to directory
