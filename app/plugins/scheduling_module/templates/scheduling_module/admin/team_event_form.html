Team event → run sheet

Creates one Time Billing run sheet for everyone (e.g. training day), with one schedule shift per person on the same date and job type. After you publish, each person gets a timesheet line; they enter actual start/end on the run sheet in the employee portal, which updates their timesheet.

Use your Training (or similar) job type so timesheets and reporting stay consistent.
Hold Ctrl / Cmd to select multiple (e.g. two trainers, one coordinator, twenty trainees).
