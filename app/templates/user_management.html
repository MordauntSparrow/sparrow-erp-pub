User Management

How this screen behaves
  • Add New User is available to you as an organisation administrator: you can create staff and crew accounts. {% if editor_may_assign_elevated %} You may also assign admin or clinical lead when creating users. {% else %} To assign admin or clinical lead, a superuser (or an account with “Core — manage administrator accounts”) must create or promote those accounts. {% endif %}
  • Permissions may look empty for admin (or similar) roles when access is implied by the role rather than a stored checklist — that is normal, not a fault.
  • Edit / Delete on other administrator accounts (admin or clinical lead) is reserved for a superuser. If you only see blank actions for those rows, that is expected — not a broken deployment.
  • Personal PIN — when you use Edit on your own row, you can set or change your 6-digit PIN (same rules as Personal PIN in the user menu). Editing someone else never shows PIN fields; admins cannot set another person’s PIN.
