# Compliance documentation pack

Use these documents to produce **evidence** for UK GDPR / DPA 2018, NHS Data Security and Protection Toolkit (DSPT), and related assurance (e.g. DTAC where applicable). They are **templates and control maps**, not legal advice.

| Document | Use |
|----------|-----|
| [CONTROL_MAP_UK_GDPR_DSPT.md](./CONTROL_MAP_UK_GDPR_DSPT.md) | Map product behaviour to common control themes |
| [EVIDENCE_CHECKLIST.md](./EVIDENCE_CHECKLIST.md) | Pre-audit / DSPT self-assessment checklist |
| [ROPA_TEMPLATE.md](./ROPA_TEMPLATE.md) | Article 30–style processing register (fill per deployment) |
| [DPIA_SCAFFOLD.md](./DPIA_SCAFFOLD.md) | High-risk processing assessment scaffold (e.g. health data) |
| [SUBPROCESSORS_REGISTER.md](./SUBPROCESSORS_REGISTER.md) | Third parties (hosting, email, error reporting, GitLab, etc.) |
| [INCIDENT_RESPONSE.md](./INCIDENT_RESPONSE.md) | Breach / incident steps including ICO notification window |
| [DATA_RETENTION_POLICY.md](./DATA_RETENTION_POLICY.md) | Retention principles by data class |
| [PRIVACY_NOTICE_TEMPLATE.md](./PRIVACY_NOTICE_TEMPLATE.md) | Public privacy information (legal review required) |
| [COOKIE_DESCRIPTION.md](./COOKIE_DESCRIPTION.md) | Cookie / similar tech description for transparency |
| [NHS_DTAC_NOTES.md](./NHS_DTAC_NOTES.md) | DTAC-style evidence pointers (NHS contexts) |
| [DCB0129_DCB0160_GUIDE.md](./DCB0129_DCB0160_GUIDE.md) | Clinical risk management: supplier vs deployer, NHS England links, Sparrow notes |
| [SOFTWARE_SECURITY_CODE_OF_PRACTICE_ALIGNMENT.md](./SOFTWARE_SECURITY_CODE_OF_PRACTICE_ALIGNMENT.md) | DSIT Software Security Code of Practice vs repo + **out-of-repo** checklist (NHS DSPT / ICO) |

**Owner:** Assign a named DPO / IG lead and version-control completed copies outside the repo if they contain sensitive organisational detail.

**Security contact template (private git):** see root [`SECURITY.md`](../../SECURITY.md) for coordinated disclosure wording to adapt for DSPT evidence or a public-facing policy page.
